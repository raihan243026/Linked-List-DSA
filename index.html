What is a Linked List?

English Explanation

A Linked List is a linear data structure where elements are not stored in contiguous memory locations. Each element (called a node) contains:

  • Data: The value
  • Next Pointer: Reference to the next node

Unlike arrays, linked lists allow dynamic size and efficient insertion/deletion.

Implementation in C++

Node Structure

// Node Definition
struct Node {
    int data;
    Node* next;
    Node(int val) : data(val), next(nullptr) {}
};

Insert at End

void insertAtEnd(Node*& head, int val) {
    Node* newNode = new Node(val);
    if (!head) {
        head = newNode;
        return;
    }
    Node* temp = head;
    while (temp->next) {
        temp = temp->next;
    }
    temp->next = newNode;
}

Display List

void display(Node* head) {
    Node* temp = head;
    while (temp) {
        cout << temp->data;
        if (temp->next) cout << " -> ";
        temp = temp->next;
    }
    cout << " -> NULL" << endl;
}
